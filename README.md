# Claude Cowork Plugins for Marketing

A curated marketplace of Claude Cowork plugins for marketing professionals.

## Why This Marketplace?

Claude Cowork is powerful, but out of the box it doesn't know your brand voice, SEO best practices, or marketing workflows. These plugins fix that.

This marketplace brings together the best community-built marketing plugins in one place:

- **SEO & Content** - Keyword research, SERP analysis, content optimization
- **Analytics** - Google Analytics 4, Search Console, DataForSEO integrations
- **Copywriting** - Brand voice, copy frameworks, headline generation
- **CRO** - Landing pages, signup flows, A/B testing, pricing strategy
- **Campaigns** - Email sequences, social content, paid ads

Each plugin is linked to its original source repository. We curate, test, and maintain compatibility—but credit stays with the creators.

## Installation

Add this marketplace to Claude Cowork desktop:

1. Open **Claude Cowork** on desktop
2. In the left sidebar, click **"Customize"**
3. Next to **"Personal Plugins"**, click the **"+"** button
4. Select **"Browse plugins"**
5. Click the **"Personal"** tab, then the **"+"** icon
6. Select **"Add marketplace from GitHub"**
7. Paste: `https://github.com/oliwoodsuk/claude-cowork-plugins-for-marketing`

Once added, you can browse and install any plugin from this marketplace directly in Claude Cowork.

## Development

### Running Tests

### Requirements

- Python 3.9+
- Node.js 18+ (for marketing-skills CLI validation)
- PyYAML and requests packages

```bash
pip3 install pyyaml requests
```

### Run Individual Plugin Tests

```bash
# Context-Link (API + structure tests)
python3 "plugins/context-link(by-context-link)/tests/test_context_link.py"

# Marketing by Anthropic (structure tests)
python3 "plugins/marketing(by-anthropic)/tests/test_marketing_structure.py"

# Marketing-Skills (structure + CLI syntax tests)
python3 "plugins/marketing-skills(by-conversion-factory)/tests/test_marketing_skills.py"

# SEO-Machine structure tests
python3 "plugins/seo-machine(by-craig-hewitt)/tests/test_structure.py"

# SEO-Machine integration tests (requires API credentials)
python3 "plugins/seo-machine(by-craig-hewitt)/tests/test_integrations.py"
```

### Run All Tests

```bash
for f in plugins/*/tests/test_*.py; do
  echo "=== $f ==="
  python3 "$f"
done
```

### Test Output

Test results are saved to markdown files in each plugin's `tests/output/` directory:

```
plugins/
├── context-link(by-context-link)/tests/output/test-results.md
├── marketing(by-anthropic)/tests/output/test-results.md
├── marketing-skills(by-conversion-factory)/tests/output/test-results.md
└── seo-machine(by-craig-hewitt)/tests/output/
    ├── test-structure-results.md
    └── test-integrations-results.md
```
